================================================================================
                    Wiring Schematic Overview
================================================================================

POWER SUPPLY CHAIN:
12V DC Power Supply ──┬──► Arduino Mega (7-12V Input)
                      │
                      ├─► Voltage Regulator (5V) ──┬──► Servos (5V)
                      │                            │
                      │                            └─► LIDAR LD06 (5V)
                      │
                      └─► Stepper Motor Driver (12V)

SERIAL COMMUNICATION:
Raspberry Pi USB ───► Arduino Mega USB ───► Serial Monitor (Debug)

CONTROL SIGNAL FLOW:
Raspberry Pi ───Serial───► Arduino Mega ───PWM───► Servo Motors
                                      ├─STEP/DIR──► Stepper Driver
                                      └─Digital───► Sensors

DATA FLOW:
Camera ──USB──► Raspberry Pi ──AI Processing──► Decision Engine
LIDAR ──Serial──► Raspberry Pi ──Distance Data──► Sensor Fusion
Sensors ──Digital──► Arduino Mega ──Serial──► Raspberry Pi

================================================================================
                    Connection Verification Checklist
================================================================================

□ Arduino Mega power connected (7-12V, 500mA minimum)
□ All servo motors connected to PWM pins (3-7) with proper power
□ Stepper driver connected to digital pins (8-13)
□ Ultrasonic sensor connected to pins 22-23
□ LD06 LIDAR connected to Serial2 pins (16-17)
□ USB cable connected between Raspberry Pi and Arduino
□ Camera connected to Raspberry Pi USB port
□ All ground connections properly connected
□ Power supply voltages verified with multimeter
□ Serial communication tested at 9600 baud rate

================================================================================
                    Troubleshooting Pin Connections
================================================================================

If servos don't move:
- Check PWM pins (3-7) are correctly connected
- Verify servo power supply (5V, adequate current)
- Ensure servo signal wires are connected to correct Arduino pins

If stepper motor doesn't work:
- Check STEP (pin 8) and DIR (pin 9) connections
- Verify A4988 driver power and enable pin
- Check microstepping pins (MS1-MS3) configuration

If LIDAR data not received:
- Verify Serial2 pins (16-17) connections
- Check LD06 power supply (5V)
- Confirm baud rate matches (230400 for LD06)

If ultrasonic sensor fails:
- Check trigger (pin 22) and echo (pin 23) connections
- Verify sensor power and ground
- Test sensor with simple distance measurement code
